# Disney API

Disney API is a rest API based on the Disney Characters.

## References

- https://swapi.dev/
- https://pokeapi.co/
- https://rickandmortyapi.com/
- https://superheroapi.com/index.html
- https://github.com/Squirrels/disney-api

## Models

- **characters**
  - id
  - name
  - fullname
  - createdBy
  - voiceBy
  - firstSeen *film
  - specie
  - alias
  - relatives *character
  - appearedOn *film
  - url
  

- **films**
  - id
  - title
  - year
  

## Endpoints
 
- `/characters`
- `/characters/id`
- `/princess`
- `/films`
